- **DefiTextMine_OpenAI.ipynb** -- ane approche utilisant ChatGPT
- **DefiTextMine_OpenAI_RAG.ipynb** -- une approche utilisant ChatGPT + RAG

- **DefiTextMine_xAI_RAG.ipynb** -- une approche utilisant Grok + RAG
